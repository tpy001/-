## DWM快捷键

`Alt + shift + Enter     # 打开新窗口`

`Alt + shift + c     # 关闭当前窗口`

`Alt + d     # 窗口横向排列`

`Alt + Enter     # 窗口位置互换`

`Alt + i     # 窗口纵向排列`

 `Alt + J  和  Alt + K # 在窗口间切换 `  

`Alt + num    # 切换标签页` 

`Alt + T     # 切换为平铺模式（tiling) `

`Alt + M    # 切换为单窗口模式`

` Alt + F   # 切换为浮动模式（float)`
