## Adobe Illustrator快捷键

常用快捷键

`H                      # 切换为抓手`

`空格不放                # 临时切换为抓手`

`空格+Ctrl不放           # 临时切换为放大镜，可放大`

`空格+Ctrl+Alt不放       # 临时切换为放大镜，可缩小`

`M                      # 矩形工具`

`\                      # 直线`

`V                      # 选择工具`

`Ctrl + O               # 打开本地文件`

`T                      # 插入文本`
